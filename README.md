# SW2627-AI-Licious-order-management

## Overview

Licious Order Management is a real-time order and inventory management system designed to help shop owners efficiently handle incoming orders without refreshing the page. The application enables seamless order tracking, live inventory updates, and maintains a complete audit trail of inventory changes.

The project is being developed as a full-stack web application, beginning with a Figma UI design and progressing through frontend, backend, and full-stack implementation.

## Problem Statement

Licious wants an order management screen where new orders appear automatically (without refreshing the page). Shop owners should be able to update order statuses (Preparing → Ready), while inventory updates instantly. Every inventory modification must be recorded with the owner ID and timestamp for accountability.

## Project Goals

* Display new orders in real time.
* Allow shop owners to manage order statuses.
* Automatically update inventory after order processing.
* Maintain an audit log for every inventory change.
* Provide a responsive, user-friendly dashboard for efficient order management.
